# aoc2025

My solutions in rust (and others) for Advent of Code 2025

## How to run

1. Add inputs in the `inputs` folder, inside a folder named `dayXX` where `XX` is the day number (e.g. `day01`, `day02`, etc.).

2. To run a specific day's solution, use the following command:

  ```bash
  cargo run --bin dayXX
  ```
